# 🚀 TaskFlow - Productivity Dashboard

TaskFlow is a modern **Productivity Dashboard** designed to help users manage their daily activities in one place.

It is an all-in-one productivity suite that includes multiple applications like Task Manager, Notes, Expense Tracker, Calendar, Quotes Generator, Weather App and Dark Mode.

The main objective of this project is to build a useful productivity platform using **HTML, CSS and JavaScript** with a clean and responsive user interface.

---

# ✨ Features

## ✅ To-Do Manager
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Store tasks using LocalStorage

## 📝 Notes App
- Create personal notes
- Save notes permanently
- Delete notes easily

## 💰 Expense Tracker
- Add income and expenses
- Calculate total balance
- Track transactions

## 📅 Calendar App
- View current month calendar
- Navigate between months
- Add and manage events

## 💡 Quotes App
- Generate motivational quotes
- Copy quotes
- Share quotes

## 🌙 Dark Mode
- Switch between light and dark themes
- Save theme preference using LocalStorage

## 🌦 Weather App
- Get real-time weather information using API integration
